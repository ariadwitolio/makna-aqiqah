## Animation Library

### Used Animations:
1. **Fade In:** opacity 0 → 1
2. **Slide Up:** y: 20 → 0
3. **Scale:** scale: 0.95 → 1 or 1 → 1.02
4. **Stagger:** Sequential child animations
5. **Spring:** Smooth bounce effect
6. **Rotate:** Logo on hover
7. **Color Transition:** Smooth color changes

### Animation Timing:
- Duration: 0.4s - 0.6s (smooth, not fast)
- Delay: 0.1s - 0.5s (staggered sequences)
- Type: ease-out (natural deceleration)
- Spring: { stiffness: 200, damping: 20 }

## Mobile Responsiveness

### Breakpoints & Changes:
```
Mobile (< 640px):
- Hero: Full width, responsive padding
- Trust badges: 2-column grid
- Services: 1 column
- Packages: Stacked (1 column)
- CTA buttons: Full width

Small (640px - 768px):
- Trust badges: 3-5 columns
- Services: 2 columns starting
- Packages: Starting to show 2-3

Desktop (> 1024px):
- Hero: 1.3fr width (more prominent)
- Highlights: 0.7fr width
- All grids: Optimal multi-column layout
- Floating button: Optimized size
```
