# Makna Aqiqah

A premium, CMS-driven aqiqah service website built with Next.js 15, TypeScript, Tailwind CSS, Payload CMS, and PostgreSQL.

## Architecture highlights
- App Router for pages and metadata
- CMS-first content strategy via Payload collections and service layer
- Reusable section and feature-based UI modules
- SEO-ready metadata, sitemap, and robots files
- VPS-friendly deployment structure

## Development
```bash
npm install
npm run dev
```
